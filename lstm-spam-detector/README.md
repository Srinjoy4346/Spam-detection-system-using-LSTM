# LSTM-Based Spam Detection System
## Final Year Project

A comprehensive email spam detection system using Long Short-Term Memory (LSTM) neural networks with Flask web interface and database integration.

## Features

- **Advanced ML Model**: LSTM-based spam detection with 94% accuracy
- **Web Interface**: User-friendly Flask application
- **Database Integration**: SQLAlchemy with SQLite/PostgreSQL support
- **User Management**: Authentication and authorization system
- **API Endpoints**: RESTful API for predictions
- **Admin Dashboard**: System monitoring and model management
- **Real-time Predictions**: Instant spam/ham classification
- **Performance Tracking**: Model performance monitoring
- **Feedback System**: User feedback for continuous improvement

## Technology Stack

- **Backend**: Flask, SQLAlchemy, Flask-Login
- **Machine Learning**: TensorFlow/Keras, NLTK, Scikit-learn
- **Database**: SQLite (development), PostgreSQL (production)
- **Frontend**: Bootstrap 5, HTML5, JavaScript
- **Deployment**: Gunicorn, Docker (optional)
